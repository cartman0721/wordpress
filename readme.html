安装：著名的5分钟安装

  1. 将压缩包解压缩到一个空目录中，并将所有文件都进行上传。
  2. 在您的浏览器中打开 wp-admin/install.php。在处理的过程中会生成一个包含您数据库连接信息的 wp-config.php 文件。
    1. 如果不能自动生成该文件，也请不要担心。不是所有主机都可以自动生成的。您可以使用文本编辑器（尽量使用 Notepad++ 等类似的软件）打开 wp-config-sample.php 文件，并在其中填写您数据库的连接信息。
    2. 将该文件保存为 wp-config.php，并将其上传。
    3. 在浏览器中打开 wp-admin/install.php 继续安装。
  3. 一旦配置文件创建完毕，安装程序将会创建您博客所需的数据库表。如果出现错误，请仔细检查 wp-config.php 文件，然后重试。如果仍然出错，可以访问支持论坛寻求帮助。
  4. 如果您没有指定密码，那么请记录下为您随机生成的密码。如果您没有指定用户名，那么默认用户名将为 admin。
  5. 然后安装程序将为您显示一个登录页面。使用您在安装过程中选择的用户名和密码进行登录。如果密码是随机生成的，您可以访问'档案'页面来进行修改。

更新

使用自动更新工具

如果您从 2.7 或更高的版本进行更新，您可以使用自动更新工具：

  1. 使用浏览器打开 wp-admin/update-core.php 并参考其中的介绍进行操作。
  2. 然后……没有然后了……就是这么简单！

手动更新

  1. 在更新任何东西之前，请先确保您对所有曾经修改过的文件进行了备份，例如对 index.php 文件进行的修改。
  2. 备份曾修改过的文件，然后删除老版本 WordPress 文件。
  3. 上传新版本文件。
  4. 使用浏览器访问 /wp-admin/upgrade.php。

从其它系统进行迁移

WordPress 可以从很多系统中进行导入。在使用我们的导入工具之前，首先需要确定您的 WordPress 已经可以正常的运行。

系统要求

  • PHP 版本 5.2.4 或以上。
  • MySQL 版本 5.0 或以上。

系统建议

  • Apache 的 mod_rewrite 模块。
  • 在您的站点中指向 http://wordpress.org 的链接。

XML-RPC 和 Atom 接口

您现在可以通过一些工具来发表文章到您的博客，比如：Windows Live Writer，Ecto，w.bloggar，Radio Userland（使用 Radio 的 email-to-blog 功能），NewzCrawler，以及其它支持博客 API 的工具。这里是更多的 XML-RPC 支持。

Post via Email

您可以从 Email 客户端发表文章！要使用这个功能，到"撰写"中填写有关您的保密的 POP3 帐户的连接信息。您需要设置 wp-mail.php ，使它可以周期性地检查您邮箱中新文章的发表。您可以通过计划任务实现，如果您的主机不支持，您可以尝试使用各种网站监视服务来检查您的 wp-mail.php URL。

发送文章很容易：任何发送到指定邮箱的文章都会被发表，以邮件主题做为文章标题。最好保密该邮件地址。成功发表文章后，程序会删除邮件。
